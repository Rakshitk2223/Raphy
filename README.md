# Raphael

A personal AI assistant inspired by the Great Sage from "That Time I Got Reincarnated as a Slime."

## Features

- Local LLM running on your machine (privacy-first)
- Hinglish (Hindi + English) support
- Beautiful galaxy-themed orb UI
- Voice and text interaction (Phase 2)
- Memory and preference learning (Phase 3)
- Web search and file browsing (Phase 4)
- System control capabilities (Phase 5)

## Quick Start

```bash
# Install
./scripts/install.sh

# Run
./scripts/start.sh
```

Then open http://localhost:8080

## Requirements

- Python 3.11+
- Ollama
- 8GB+ VRAM (RTX 5050 or similar)
- 16GB RAM
